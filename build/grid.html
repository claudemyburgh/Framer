Structure your html to create a mobile first grid layout

By default Framer is a 12 column grid, but this can easaliy be changed to a 16 column grid in seconds. css classes can be stacked to create multipale layout for diffrand devices without writing and additional code.

<!--Create a simple grid without gutters-->
<div class="row">
  <div class="sm-col-12 md-col-6 lg-col-3 xl-col-2">C 1</div>
  <div class="lg-col-2">Col 2</div>
  <div class="lg-col-3">Column 3</div>
  <div class="lg-col-4">Column 4</div>
  <div class="lg-col-2">Col 2</div>
</div>
